# Семейный Бюджет

Небольшой веб‑проект с приветственным лендингом и калькулятором семейного бюджета.

## Что внутри

- `index.html` — лендинг проекта.
- `калькулятор.html` — страница с основным функционалом.
- `404.html` — страница ошибки для несуществующих маршрутов.
- `assets/css/styles.css` — общие стили.
- `assets/js/main.js` — базовый JS для общих элементов.
- `assets/js/calculator.js` — логика калькулятора.
- `assets/js/calculator-core.js` — переиспользуемые расчетные функции.
- `assets/favicon.svg` — favicon проекта.
- `robots.txt` и `sitemap.xml` — SEO-файлы.
- `tests/calculator.test.js` — базовые автотесты расчетов.
- `package.json` — скрипты проекта.

## Возможности калькулятора

- Добавление и удаление статей доходов и расходов.
- Расчет баланса, доли сбережений, сумм на одного человека.
- Настройка 2 целей сбережений: минимальная и комфортная.
- Процент выполнения целей и прогресс-бары.
- Мини‑график структуры расходов по категориям.
- Переключение светлой/темной темы.
- Экспорт и импорт данных в JSON.
- Автосохранение в `localStorage`.

## Тесты

```bash
npm test
```

## Важно перед публикацией

- В файлах `robots.txt` и `sitemap.xml` замените `https://example.com` на ваш реальный домен.

## Как запустить

1. Откройте файл `index.html` в браузере.
2. Нажмите кнопку «Открыть калькулятор» или перейдите в `калькулятор.html`.
